                                     Data Dashboard Patch Notes
                                              2023-07-21


Market Scan
      The Market Scan has undergone significant changes both with its UI and its quarriable metrics.

      UI
      - The selections are now located at the top of the page, and can be accessed by pressing the
         relevant button.
      - The market scan is initiated via pressing “Perform Scan” after selecting all of the desired
         selection criteria.
      - The selection criteria can now be reset with the “Reset Scan” button.
      - Several “pop-up” tool tips have been added to test their functionality and benefit.


      Functionality

      -   Directional Options Filter
                  The Market Scan can now filter for
                       Dealer Long / Short OTM Calls
                       Dealer Long / Short OTM Puts
                  There are two additional criteria that can be selected within the “Directional Options”
                  criteria:
                       Remove Outliers : this will remove the top and bottom 25% off the results,
                           based on the percent change.
                       “Set %-Change Ranges” : if a particular greater-than or less-than or range-of-
                           percent-change values are desired, this can be selected, and the relevant inputs
                           provided.

      -   Volume Filter
                Filtering by Volume can now be done in a similar way as described above, with greater-
                than, less-than, or a range of values being selected.

      Future Work
      -   Future work will be aimed at increasing the dynamic nature and useability of the market scan
          for the currently available criteria as well as adding some more.

General Updates
   -   Approximately 1,100 new stocks have been added to DDS’ repertoire. For a complete list of
       added stocks, see the last page.


Bug Fixes

   -   There was a bug causing erroneous error handling due to certain inputs when searching for stock
       data. This has been fixed.
   -   A bug causing volume to not be appropriately filtered-for in the Market Scan. This has been fixed.
   -   There were issues with some unintended files being transferred to the Dashboard as report
       PDFs. This has been fixed.
   -   There was an issue with checking user-supplied inputs over-reacting. This has been fixed.
